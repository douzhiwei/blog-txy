单线程

要说js异步操作首先还的从js的单线程说起

  • 单线程: 只有一个线程，只能做一件事
  • 原因: 避免 DOM 渲染的冲突
    • 浏览器需要渲染 DOM
    • JS 可以修改 DOM 结构
    • JS 执行的时候，浏览器 DOM 渲染会暂停
    • 两段 JS 也不能同时执行（都修改 DOM 就冲突了）
    • serviceworker 支持多线程，但是不能访问 DOM
