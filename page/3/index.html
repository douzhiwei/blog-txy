exec,match,replace,test方法对比

发表于 2018-03-09 | 分类于 js易忘知识点

注：pattern 为RegExp的实例, str 为String的实例

用法 说明 返回值
pattern.test(str) 判断str是否包含匹配结果 包含返回true，不包含返回false
pattern.exec(str) 根据pattern对str进行正则匹配 返回匹配结果数组,如匹配不到返回null
str.match(pattern) 根据pattern对str进行正则匹配 返回匹配结果数组,如匹配不到返回null
str.replace(pattern, replacement) 根据pattern进行正则匹配,把匹配结果替换为replacement 一个新的字符串

云服务器安装node环境

发表于 2018-03-07 | 分类于 Node

  1. 先更新一下系统
    sudo apt-get update

  2. 安装一些常用的工具
    sudo apt-get install vim openssl build-essential libssl-dev wget curl git

  3. 安装nvm
    wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

安装完毕会提示再打开一个控制台

  1. 使用nvm安装node,知道版本号,我这安装的是7
    nvm install vv8.9.1

  2. 使用vv8.9.1为默认node版本
    nvm use vv8.9.1
    nvm alias default vv8.9.1

  3. 使用淘宝镜像cnpm
    npm install -g cnpm --registry=https://registry.npm.taobao.org

ok,node环境搭建完毕
