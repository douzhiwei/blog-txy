css3-css3边框

发表于 2018-07-18

css2边框属性比较简单有3种

  • border-width 设置边框的粗细
  • border-color 设置边框的颜色
  • border-style 设置边框的演示

css3边框新增属性如下

  • border-image 设置边框背景图
  • border-radius 设置边框的弧度
  • box-shadow 设置边框阴影,可以做出非常好看的效果

border-image

border-image: none | <img> [<number> | <percentage>] {1,4} [/ <border-width>{1,4}] ? [stretch | repeat | round] {0, 2}

  • none 默认值,表示边框无背景图片
  • <image> 设置背景图片,和background-image一样
  • <number>设置边框或背景图片的大小,单位是px,可以设置1~4个值,表示4个方位
  • <percentage>跟number用法一样,只不过是百分比
  • stretch , repeat , round设置图片的铺放方式, stretch是默认值
    • stretch拉伸图片
    • repeat重复背景
    • round 平铺

border-radius

border-radius: none | {1, 4}
这个参数的属性还是挺多的,但是都不是很常用,可以制作圆形,扇形,不规则的角度,不展开说了

box-shadow

box-shadow是c3新增的一个重要属性,用来定义盒子的阴影,我认为不加这个属性不会影响什么但是加了会提升用户体验,详细看一下

box-shadow: none | [inset x-offset y-offset blur-radius spread-radius color]

  • none 默认值,没有任何的阴影效果
  • inset 阴影类型,可选,默认值是外阴影,如果设置inset 则为内阴影
  • x-offset 阴影水平偏移量,如果是正值,阴影在元素右边,如果是负值,阴影在元素左边
  • y-offset 阴影垂直偏移量,如果是正值,阴影在元素底部,如果是负值,阴影在元素的顶部
  • blur-radius 阴影模糊半径,可选值,只能是正值,如果为0,不具有模糊效果,取值越大,边缘越模糊
  • spread-radius 阴影扩展半径,可选参数,如果为正值,整个阴影延展扩大,为负值,阴影缩小
  • color 阴影的颜色
